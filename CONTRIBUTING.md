# Contributions Welcome!

To contribute, please do the following:
1. [Fork the Repository](https://docs.github.com/en/get-started/quickstart/fork-a-repo) into your personal GitHub or organization
2. Add your own changes to your forked repo
3. If you added changes to another branch, make sure to [merge it into your repo's main branch](https://git-scm.com/book/en/v2/Git-Branching-Basic-Branching-and-Merging#_basic_merging)
4. Create a [Pull Request](https://docs.github.com/en/pull-requests/collaborating-with-pull-requests/proposing-changes-to-your-work-with-pull-requests/creating-a-pull-request-from-a-fork)
    1. Make sure to clearly explain the change in the PR!
